# com-sci-ia
This C program was made for my CAPE Computer Science Internal Assessment for the 2025-2026 academic year.

Prompt:
This program will take in the ID number given to the book, as well as the name and the ID number of the person renting. Upon return, how many days the book was outstanding will be inputted in addition to the previously stated information to determine if the book was returned on time or not, and calculate an appropriate late fee based on how many days late the book is. Additionally, it will generate a report on books and their renters’ information sorted by returned, rented, if the book is overdue and by how many days. If a person is late with their return they receive a warning on the system, and after three warnings the person will not be allowed to rent again. Moreover, persons who have not returned a rented book will not be permitted to rent another book unless the first is returned.
